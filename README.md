# FitPulse Lite (No-PC APK via GitHub Actions)

Super light fitness tracker:
- Daily goal (minutes)
- Simple timer
- Quick workout logs
- Offline storage with SharedPreferences (fast build)

## Get an APK without a PC (Phone only)
1) Upload this project to a GitHub repo (name it anything, e.g. FitPulseLite).
2) Go to repo → Actions → "Build APK (Lite)" → Run workflow.
3) Download artifact "FitPulseLite-debug-apk" → extract → app-debug.apk
4) Tap APK on your Honor 90 → Package Installer → Install

If blocked: Settings → Security → Install apps from external sources → allow Files/Browser.
